Creates an "Author Box" Widget with the author's picture, bio and social media links.

== Description ==

This plugin will create an authorbox widget that you can place in a sidebar. The widget displays the users gravatar, biographical info and icons linking to Facebook, Twitter, LinkedIn and Google Plus.

Edit the users profile page and provide the necessary information. The social media links should be full URL:s, i e 
http://www.facebook.com/joe-smith
http://www.twitter.com/joe-smith

== Installation ==

1. Upload Social Author Box to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add social media profile links on the user profile page
